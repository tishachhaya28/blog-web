# Blog Web 🚀

A modern, full-stack blog platform built with React, Node.js, and MongoDB.

## ✨ Features
- **Admin Dashboard**: Secure post management.
- **RBAC**: Role-based access control (Admin/User).
- **Premium Ads**: Integrated Google AdSense slots (Banner, Side, Footer).
- **Responsive**: Fully responsive design with Tailwind CSS v4.

## 🛠 Tech Stack
- **Frontend**: React 19, TypeScript, Vite, Tailwind v4, Shadcn.
- **Backend**: Node.js, Express, MongoDB, JWT.

## 🏁 Quick Start

### 1. Backend
```bash
cd backend
npm install
npm start
```
*Requires `.env` with `MONGO_URI`, `PORT`, and `JWT_SECRET`.*

### 2. Frontend
```bash
cd frontend
npm install
npm run dev
```
*Requires `.env` with `VITE_BASE_URL`.*

## 📈 Deployment
- **Frontend**: Vercel
- **Backend**: Render
- **Database**: MongoDB Atlas
